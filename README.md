##野人传教士过河

2018年6月1日

设有3个传教士（Missionaries）和3个野人(Cannibals)来到河边，打算乘一只船从右岸渡到左岸去。
该船的最大负荷能力为两个人（k=2）。在任何情况下：如果野人人数超过传教士人数，那么野人就会把传教士吃掉。
他们怎样才能用这条船安全地把所有人都渡过河去呢？
